# Python Operators

## Bitwise

| Operation                | Syntax | Function        |
| ------------------------ | ------ | --------------- |
| bitwise and              | &      | operator.and\_  |
| bitwise or               | \|     | operator.or\_   |
| bitwise xor              | ^      | operator.xor    |
| bitwise not (or inverse) | \~     | operator.invert |
